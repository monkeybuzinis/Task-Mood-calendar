## Description
A task management application using React and Express

## Tech Stack
- Frontend: React, Axios
- Backend: Node.js, Express, MongoDB

## Getting Started

### Backend Setup
```bash
cd backend
npm install
node server.js
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
